1. Click Wamp, Apache -> Apache Modules -> enable rewrite_module (might be called mod_rewrite)
	Need to enable the following PHP extensions:
		php_curl,
		php_openssl.
2. Add the following to "C:\wamp\bin\apache\apache2.4.2\conf\extra\httpd-vhosts.conf":

<VirtualHost *:80>
    ServerAdmin webmaster@dummy-host.example.com
    DocumentRoot "c:/wamp/www/silex/web"
    ServerName api.geolabel.local
    ServerAlias www.api.geolabel.local
    ErrorLog "logs/geolabel.local-error.log"
    CustomLog "logs/geolabel.local-access.log" common
</VirtualHost>

3. Uncomment the following in "C:\wamp\bin\apache\apache2.4.2\conf\httpd.conf":
Include conf/extra/httpd-vhosts.conf (just remove # sign)

4. Run Notepad as Administrator, open "C:\Windows\System32\drivers\etc\hosts" and add the following:

127.0.0.1	api.geolabel.local
127.0.0.1	www.api.geolabel.local

5. Restart Wamp
